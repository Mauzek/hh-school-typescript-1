# 🛠 Утилитарные функции: JavaScript & TypeScript

В данном репозитории реализованы кастомные версии популярных утилитарных функций. Проект демонстрирует понимание асинхронности, замыканий, работы с контекстом (`this`), метапрограммирования (`Proxy`/`Reflect`) и продвинутой типизации (Advanced TypeScript).

## 📦 Реализованные функции

1. **`promiseAll(promises)`**
   * Кастомная реализация `Promise.all`.
   * Сохраняет строгий порядок возвращаемых результатов независимо от времени выполнения промисов.
   * Немедленно отклоняется (fail-fast) при первой ошибке.
   * Безопасно обрабатывает пустые массивы и примитивы (не-промисы).

2. **`delay(ms)`**
   * Асинхронная пауза на базе промисов. Идеально для использования с `async/await`.

3. **`memoize(fn)`**
   * Функция высшего порядка для кэширования результатов вычислений.
   * Отличает типы данных в аргументах (например, число `1` и строку `"1"`).
   * 🛡 **Особенность:** Корректно сохраняет и пробрасывает контекст вызова (`this`), что позволяет безопасно оборачивать методы объектов.

4. **`typedObject(schema)`**
   * Создает объект со строгой проверкой типов в рантайме (во время выполнения) на базе `Proxy` и `Reflect`.
   * Бросает `TypeError` при попытке записи неверного типа или несуществующего свойства.
   * 🛡 **Особенность:** Защищено от падений при внутренних вызовах движка JS (корректно игнорирует `Symbol`).
